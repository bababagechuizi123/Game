C++ developed 2D survivor like game
==========
A 2D game developed using GamesEngineeringBase framework.
----------

The game includes two levels with different maps.
During the game, bullets will shoot from the player automatically, chasing the closest enemy. Player also has
a active AOE skill, that can attack the top 3 max health
enemies inside a certain range. after killing enough
enemies, hero will gain a powerup that increases the
speed of the bullets and number of enemies targeted
by the AOE skill. There are 4 types of enemies: mushroom enemies with the color red, green and purple
respectively, and a flying bat enemy. the mushroom
will chase the player automatically, and can only attack
when they are close enough with the player. mushrooms are differed by different HP, speed and attack
desire. The bat will attack from a distance, creating
a cyclone towards the player. Killing different type of
enemy will gain different score, which will be shown at
the top left corner of the screen through the game. The
game will be saved automatically at any point when
quitting, player can choose load the save or start a new
game as they like.
